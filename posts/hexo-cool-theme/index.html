2021/4/19[（HEXO炫酷主题）]{.rainbow}

+++primary 点击展开
:::info
安装
:::

  • npm un hexo-renderer-marked –save
  • npm i hexo-renderer-multi-markdown-it –save

+++

如果安装缓慢，或者失败

  • 如报错

  • 因为有一步需要下载 puppeteer 里的 Chromium 内核，基于天朝内部网络现状，这一步能不能成功要靠科学和运气，所以为了避免安装失败，需要加上 –ignore-scripts 跳过 Chromium 内核的下载。

  • npm i hexo-renderer-multi-markdown-it –save –ignore-scripts

  • puppeteer 主要是用来渲染 mermaid 流程图，只要文章中不使用 mermaid 就没有任何问题，如果要使用 mermaid 建议还是想办法完全安装。

+++danger 点击展开
[安装]{.label .danger}
-npm install hexo-autoprefixer –save
-npm install hexo-algoliasearch –save
+++

如果不是这种情况，请在您的_config.yml文件中注册插件：

+++info 点击展开
;;;id3 安装
npm install hexo-symbols-count-time
;;;

;;;id3 继续
继续
;;;
+++

您可以在Hexo_config.yml（位于博客的根目录中）中设置hexo-symbols-count-time的选项：

[:heavy_check_mark:npm install hexo-feed –save-dev]{.label .success}

{安装^*❤} [npm uninstall js-yaml]{.kbd} + [npm install js-yaml@3.14.1]{.kbd .red}
